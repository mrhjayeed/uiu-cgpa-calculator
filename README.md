# UIU CGPA Calculator

A simple web-based GPA and CGPA calculator based on [United International University's](https://www.uiu.ac.bd/) official grading policy.

## Features

- Calculate current trimester GPA
- Calculate cumulative CGPA with previous record
- Retake course support with grade replacement
- Auto-save to local storage
- Responsive design

## Grading Scale

| Grade | Point | Marks (%) |
|-------|-------|-----------|
| A     | 4.00  | 90 – 100  |
| A-    | 3.67  | 86 – 89   |
| B+    | 3.33  | 82 – 85   |
| B     | 3.00  | 78 – 81   |
| B-    | 2.67  | 74 – 77   |
| C+    | 2.33  | 70 – 73   |
| C     | 2.00  | 66 – 69   |
| C-    | 1.67  | 62 – 65   |
| D+    | 1.33  | 58 – 61   |
| D     | 1.00  | 55 – 57   |
| F     | 0.00  | 0 – 54    |

## Usage

Open `index.html` in a browser. No build tools or dependencies required.
